Installation Guide

Set Up in Minutes

Follow these step-by-step instructions to install News Guard on your preferred browser. No coding required.

Installing on Google Chrome

Works on Chrome v88+ · Windows, macOS & Linux

1
Download the Extension Package
Download the News Guard .zip file using the download button above. Extract the ZIP to a permanent folder on your computer — do not delete this folder after installation, Chrome needs it.
💡Create a folder like C:\Extensions\NewsGuard (Windows) or ~/Extensions/NewsGuard (Mac/Linux) so it stays organized.
2
Open Chrome Extensions Page
Open Google Chrome and navigate to the extensions management page:
URL →chrome://extensions
Or go to ⋮ Menu → More Tools → Extensions.
3
Enable Developer Mode
In the top-right corner of the extensions page, you'll see a toggle labeled "Developer mode". Turn it ON. Three new buttons will appear at the top.
4
Load the Extension
Click "Load unpacked" (top-left). Navigate to and select the extracted News Guard folder (the one containing manifest.json). Click "Select Folder".
⚠️Select the folder itself, not the ZIP file or individual files inside it.
5
Pin & Start Using News Guard
News Guard is installed! Click the 🧩 puzzle icon in your Chrome toolbar, then click the 📌 pin icon next to News Guard. Open the extension to configure your alert preferences and prop firm rules.
✅You're all set! News Guard will now alert you before any high-impact news event.

Installing on Microsoft Edge

Works on Edge v88+ (Chromium-based) · Windows & macOS

1
Download & Extract the Package
Download the News Guard .zip file and extract it to a permanent folder. Keep this folder — Edge references it every time the browser starts.
2
Open Edge Extensions Page
Open Microsoft Edge and navigate to:
URL →edge://extensions
Or go to ⋯ Menu → Extensions → Manage Extensions.
3
Enable Developer Mode
Find the "Developer mode" toggle in the bottom-left sidebar. Turn it ON.
📍Edge places the toggle in the bottom-left, unlike Chrome which has it top-right.
4
Load Unpacked Extension
Click "Load unpacked" at the top. Browse to and select the extracted News Guard folder (containing manifest.json). Click "Select Folder".
5
Pin & Configure
Click the 🧩 Extensions icon in Edge's toolbar, then click the 👁 eye icon next to News Guard to show it permanently. Open it to configure your preferences.
✅Done! News Guard is now protecting your trades on Microsoft Edge.

Installing on Brave & Opera

Chromium-based browsers · Same process with minor URL changes

1
Download & Extract
Download the News Guard .zip file and extract it to a permanent folder. Both Brave and Opera are Chromium-based, so the extension works identically.
2
Open Extensions Page
Navigate to your browser's extension page:
Brave →brave://extensions
Opera →opera://extensions
3
Enable Developer Mode
Toggle "Developer mode" ON — located in the top-right corner (same as Chrome).
4
Load Unpacked → Select Folder
Click "Load unpacked", select the extracted News Guard folder, and confirm.
5
Pin & You're Done
Pin News Guard to your toolbar from the extensions menu and start trading safely.
🛡️Brave Shields will not interfere with News Guard. It works perfectly alongside all Brave privacy features.

Installing on Firefox

Firefox v109+ · Temporary add-on for development / testing

1
Download the Firefox Version
Download the News Guard package. Ensure you have the Firefox-compatible version (with manifest.json using "manifest_version": 2). Extract to a folder.
2
Open Firefox Debugging Page
Open Firefox and navigate to:
URL →about:debugging#/runtime/this-firefox
3
Load Temporary Add-on
Click "Load Temporary Add-on…". Navigate to your extracted News Guard folder and select the manifest.json file directly.
⚠️In Firefox, select the manifest.json file itself — not the folder like in Chrome/Edge.
4
Verify Installation
The extension should appear in the list on the debugging page with its version number and an "Inspect" button. The icon will also appear in your toolbar.
5
Important: Temporary Add-on Note
Configure your alerts and start trading. News Guard will work perfectly throughout your current session.
🔄Note: Firefox temporary add-ons are removed when you close the browser. You'll need to re-load each restart. The signed .xpi version will be available soon on Firefox Add-ons.
